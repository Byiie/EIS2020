# FQA
* Q:我的世界游戏启动失败咋办？
  * A:下载更新的版本。
* Q:我的世界多人游戏选择了localhost,一直连不进去？
  * A:确保start.bat已启动，下载游戏勾选下载forge,再不行更新java。
* Q:在将3D模型导入我的世界程序运行时出现no moudle binvox_rw？
  * A:要将.py文件与binvox文件放在一起
